# nightwatch_sample

## 安裝教學
   
1. 開啟命令列[cmd]下 `npm install -g nightwatch`

2. 執行 `git clone [shgit 儲存庫位址]`

3. 進入clone下來的資料夾執行 `npm install`

4. 如何執行測試： `npm run chrome`
